# Practica de Automatizacion con Cypress

Proyecto de prácticas centrado en la automatización de pruebas E2E con Cypress sobre la web de pruebas [SauceDemo](https://www.saucedemo.com/).

## Objetivo

El objetivo de este proyecto es practicar y afianzar conocimientos sobre:

- Automatización de pruebas end-to-end
- Uso de Cypress
- Page Object Model (POM)
- Uso de fixtures para datos de prueba
- Validación de casos positivos y negativos
- Generación de reportes HTML de ejecución

## Tecnologías utilizadas

- JavaScript
- Cypress
- Node.js
- cypress-mochawesome-reporter
